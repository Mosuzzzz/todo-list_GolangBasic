# Go CLI Todo App

Simple Todo CLI written in Go.

## Project URL
https://github.com/Mosuzzzz/todo-list_GolangBasic

## Features

- Add task
- Update task
- Change status
- Soft delete
- List tasks

## Example

add "Buy milk"
update 1 "Buy coffee"
status 1 done
list

## Installation

go run main.go

## Commands

add "task"
update 1 "task"
status 1 done
delete 1
list
